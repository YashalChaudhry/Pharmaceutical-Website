Medicine App

Welcome to the Medicine App. This application allows you to manage your medicines, including storing images in MongoDB and updating prices.

Getting Started

First, you need to sign up or log in to access the admin panel where you can manage your medicines.

Login Sign Up

Medicine Management

After logging in, you can:

  • View all medicines
  • Add new medicines with images
  • Edit existing medicines (update both images and prices)
  • Delete medicines
  • Import medicines from your assets folder to MongoDB
