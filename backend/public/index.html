Medicine API Documentation

Welcome to the Medicine API. This API allows you to manage medicine information including images and prices.

Available Endpoints

GET /api/medicines

Get all medicines from the database.

GET /api/medicines/:id

Get a single medicine by its ID.

POST /api/medicines

Create a new medicine with image.

// Example request body (multipart/form-data):
{
  "name": "Medicine Name",
  "price": 19.99,
  "description": "Medicine description",
  "category": "Pain Relief",
  "image": [file upload]
}

PATCH /api/medicines/:id

Update a medicine (including image and price).

// Example request body (multipart/form-data):
{
  "price": 24.99,
  "image": [file upload]
}

DELETE /api/medicines/:id

Delete a medicine.

POST /api/medicines/import-from-assets

Import medicines from assets folder.

API Status

GET /api/status

Check if the API is running properly.
